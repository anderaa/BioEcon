This folder contains the code for the all of the analyses. There is also a directory (HDSS_r1-12_csv) with five .csv data files that were directly copied from Anne's original Access data file.

Clicking on the Rstudio file will open Rstudio. You can then open whichever scripts you want.

Outline of Scripts:

1. Exit Analyses
	A. constant-only, with separate models for juveniles and adults (adults >= 300 days),
	   then take results and convert to annual probabability of mortality for juveniles and adults

2. Reproduction Analyses
	A. average litters per female per year
	B. average litter size

3. Other Analyses
	A. Average abundance, maximum abundance, average percent juvenile, average percent female
	B. Number of immigrant dogs per year
