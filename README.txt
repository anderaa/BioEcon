Repository Contents:
1) Shiny - this is the directory of Shiny web application.
2) Notebooks - a directory containing the Jupyter Notebook implementation of the model.
3) Scripts - a directory containing the primary script-based implementation of the model as well as scripts used to replicate results in the manuscript.
4) HDSS_Analysis - a directory containing the analysis that estimates parameters from the HDSS that are subsequently used by the shiny app.

Notes:
- The current version of the model can be accessed from www.bioeconmodel.com
- The model has been tested with R version 3.5.2
