This is the main directory of the BioEcon project.

Contents:

1) Shiny - this is the directory of Shiny web application.

2) Manuscript - this is the directory of the manuscript describing the project.

3) HDSS_Analysis - this is the directory of the analysis that estimates parameters from the HDSS that are subsequently used by the Shiny App.

4) Misc - this directory holds data sources and other project materials.
